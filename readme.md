用WinForm写的计时器。可以设置多个计时器。

输入时间的方式：

* 1-2位数代表分钟
* 3-4位数代表分钟:秒
* 5+位数代表代表小时:分钟:秒。最多输入8位。

数字后面的部分是计时器名称，如果名称本身是数字，则在数字前加空格，如`3 3分钟`表示设置一个3分钟的倒计时，计时器名称是`3分钟`。
